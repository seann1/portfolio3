A rails app that emulates some of the functionality of Stack Overflow. Users can sign up and sign in, ask questions, answer questions, vote answers up, pick the best answer to their questions and view profiles of other users. Profile pictures are hosted on Amazon s3 servers.

A webpage made with HTML, CSS and JavaScript that demonstrates Object Oriented JavaScript.
